# UK-STUDY-VISA-CHATBOT
UK STUDY VISA CHATBOT describes the queries of any person related to uk study visa process key points.
This chatbot clears the whole roadmap of uk study visa process.
This chatbot is integerated to a visa website.
This chatbot consists of manual trigger node.
This chatbot consists of https request node which fetches the uk study visa related informations from official UK govenment visa website.
Html node is also used.
Ai agent, open ai chat model and simple memory is also used.
